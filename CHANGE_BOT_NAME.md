# How to Change Bot Name

**Date:** February 2, 2026  
**Bot:** @pikotheservicedog_bot

---

## Change Bot Name via @BotFather

### Step 1: Open BotFather
1. Open Telegram
2. Search for: `@BotFather`
3. Start a chat

### Step 2: Select Your Bot
Send: `/mybots`

Select: `pikotheservicedog_bot` (or whatever it shows)

### Step 3: Choose What to Change

**Option A: Change Display Name**
1. Select: `Edit Bot` → `Edit Name`
2. Enter new name (e.g., "ClawFriend" or "Piko")
3. Confirm

**Option B: Change Username**
1. Select: `Edit Bot` → `Edit Username`
2. Enter new username (must end in `bot`, e.g., `clawfriend_bot`)
3. **Note:** If username is taken, try variations

**Option C: Change Description**
1. Select: `Edit Bot` → `Edit Description`
2. Enter description (what users see when they start the bot)
3. Example: "A witty AI assistant powered by Ollama"

**Option D: Change About Text**
1. Select: `Edit Bot` → `Edit About`
2. Enter short bio (shown in bot profile)

- Changes take effect immediately
- No need to restart the bot service
